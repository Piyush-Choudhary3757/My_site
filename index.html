Enterprise ETL Pipeline Automation

Built automated ETL pipelines using SQL and Python to ingest, cleanse, validate, and integrate 500K+ enterprise utility records from Oracle and DB2 systems into PostgreSQL. Implemented automated reconciliation checks.
Architecture: Python (Pandas/NumPy) ETL scripts scheduled via Cron, extracting data from DB2/Oracle via ODBC into PostgreSQL data warehouse.
Scaling: Implemented chunked processing for 500K+ records. Used vectorized pandas operations to reduce runtime by 40%.
SQL Logic: WITH RankedData AS (SELECT *, ROW_NUMBER() OVER(PARTITION BY id ORDER BY update_ts DESC) as rn FROM raw) SELECT * FROM RankedData WHERE rn = 1;
